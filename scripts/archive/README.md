## 再利用方法

アーカイブされたスクリプトを再度使用する場合：

```powershell
# スクリプトをアーカイブから実行
.\scripts\archive\download-phosphor-icons.ps1

# または、アクティブディレクトリに戻す
Move-Item scripts/archive/<script-name>.ps1 scripts/
```
