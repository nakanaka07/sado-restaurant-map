### 🎯 SCRAP 原則準拠基準

| 原則           | 基準   | チェックポイント           |
| -------------- | ------ | -------------------------- |
| **Specific**   | 具体的 | コード例、明確な手順       |
| **Concise**    | 簡潔   | 不要な説明の排除           |
| **Relevant**   | 関連性 | 対象読者に適した内容       |
| **Actionable** | 実用性 | 実行可能な手順             |
| **Practical**  | 実践的 | 実際の使用場面に即した内容 |
